Arrays vs. Hashes

What are they?

our_array = [ 3, "cat", true, [], {} ]

An array is a list(or collection) of objects/data that is ordered by indexed-position. The index of the first object in an array is 0 because computers start counting from zero! You can access the objects of an array through it's position, for example our_array[1] would return "cat". Arrays can hold just about anything you throw at it integers, booleans, strings, hashes even other arrays.

our_hash = { "cat" => 'meow', "dog" => 'woof' }

Hashes store data in the form of key-value pairs, unlike arrays' indexed-position hashes are accessed through their keys. You're free to assign key and values to anything you like, for example 'color' => 'orange' or 'food' => 'chicken wings' }
