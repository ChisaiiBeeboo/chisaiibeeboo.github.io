Ruby Instance Variables

In Ruby everything is an object and every object belongs to a class. Each class has their own attributes which are stored in instance variables and behaviors defined by methods. Instance variables are instantiated in the class' initialize method.

 
          
              class Steakhouse_Review
                def initialize(name, cut, rating)
                  @name = name
                  @cut = cut
                  @rating = rating
                end

                def review
                  p "#{@name}'s #{@cut} is #{@rating}."
                end
              end 

               outback = Steakhouse_Review.new("Outback","T-Bone", "not so great")
               peter_luger = Steakhouse_Review.new("Peter_Luger", "Porterhouse", "over-rated")
               delmonico = Steakhouse_Review.new("Delmonico", "Ribeye", "delicious")

               p outback.review # => "Outback's T-Bone is not so great."
               p peter_luger.review # => "Peter_Luger's Porterhouse is over-rated."
               p delmonico.review # => "Delmonico's Ribeye is delicious."

In our example we have the class Steakhouse_Review. Each time a new review(instance) of a steakhouse is generated it will contain the steakhouse name, the cut of steak and it's rating. Steakhouse_Review has one method which return that instances' attributes.
