Why is JavaScript so popular in web development?

Google happened. Gmail and Google Maps is itengral to JavaScripts popularity. Google put Javascripts potential on center stage for developers all over the world to see when they launched Gmail in April 1, 2004 and Maps on Feb 8, 2005. JavaScript transforms static webpages to dynamic webapps making the web a faster and smoother experience for users.

Improvement in hardware help computers process JS. Increase in home connection speeds. Popularity of mobile browsing. Faster interpreters made reading and executing source code instant.

Lack of competition makes Javascript the obvious language when writing a web application that works across all platforms. It's enabled by default by most browsers.
